Clipper

Background application for everyday use.

Download .zip Download .tar.gz View on GitHub

Features:

  • Easily shoot and share screenshots with ImageShack.
    Screenshots can also be saved into specified local directory.
  • Tired of long URLs? Shorten links with Google Link Shortener.
  • Rapidly share text or code snippets with TinyPaste.
  • Quickly convert anything possible into QR codes via QRickit.
  • Copy multiple text pieces, which will be automerged and placed into clipboard.
  • Track and save your clipboard history. You will never lose something important!

Platforms:

  • Linux
  • Windows

Prebuild binaries:

  • GitHub Releases

Building from source code:

Run following commands from 'Clipper/src' directory.

  • ./configure
  • make
  • sudo make install
