Clipper

Фоновое приложение для повседневного использования.

Скачать .zip Скачать .tar.gz GitHub

Особенности:

  • Создавайте и публикуйте скриншоты при помощи ImageShack.
    Скриншоты автоматически сохраняются в указанную локальную папку.
  • Устали от длинных ссылок? Сделайте их короче с Google Link Shortener.
  • Мгновенно делитесь отрывками текста или кода с помощью TinyPaste.
  • Превращайте что угодно в QR коды с QRickit.
  • Копируйте отрывки текста и соединяйте их при помощи режима Multicopy.
  • Отслеживайте и храните историю буфера обмена. Вы никогда не потеряете в нем что-то важное!

Платформы:

  • Linux (пакеты доступны для Debian и Ubuntu)
  • Windows XP и новее

Готовые сборки:

  • Релизы на GitHub

Ручная сборка из исходных кодов:

Запустите следующие команды из директории 'Clipper/src'

  • ./configure
  • make
  • sudo make install
